# UP Police 2026 — Daily Affairs Telegram Bot

Daily current affairs + practice MCQs Telegram bot for UP Police Constable 2026 exam aspirants. Source: GKToday RSS. MCQs auto-generated by Claude API.

## Features

- 📰 Daily current affairs digest at user-chosen time (IST)
- 🧠 Auto-generated practice MCQs (UP Police difficulty level)
- ⚙️ Per-user customization: time, news count, MCQ count, pause/resume
- 🎯 On-demand `/today` and `/quiz` commands
- 💾 JSON-file storage (zero-DB simplicity)

## Quick Start (Local Dev)

```bash
# 1. Clone and install
cd up-police-bot
npm install

# 2. Setup env vars
cp .env.example .env
# Edit .env — add TELEGRAM_BOT_TOKEN and ANTHROPIC_API_KEY

# 3. Run in dev mode
npm run dev

# Or build + run production
npm run build
npm start
```

## Getting the Tokens

### Telegram Bot Token

1. Open Telegram → search `@BotFather`
2. Send `/newbot`
3. Pick name (e.g. "UP Police 2026 Affairs") and username (must end in `bot`, e.g. `up_police_2026_bot`)
4. Copy the token → paste into `.env` as `TELEGRAM_BOT_TOKEN`

### Anthropic API Key

**Important:** Claude Pro subscription ($20/mo) gives you access to Claude.ai web/desktop and Claude Code, but the **API has separate billing** at [console.anthropic.com](https://console.anthropic.com).

1. Go to https://console.anthropic.com → Settings → API Keys
2. Add ~$5–10 credits (will easily last the full 40 days for this bot's MCQ usage)
3. Create key → copy → paste into `.env` as `ANTHROPIC_API_KEY`

> **Cost estimate:** Each MCQ batch (~5 MCQs) uses Claude Haiku 4.5 = roughly ₹0.10–0.30 per day. For 40 days = under ₹15 total.

> If you skip the API key, the bot still works — news digest will send, MCQ generation just gets skipped.

## Deploy to Railway

1. Push this code to a private GitHub repo
2. Go to [railway.app](https://railway.app) → New Project → Deploy from GitHub
3. Select your repo
4. In Variables tab, add:
   - `TELEGRAM_BOT_TOKEN`
   - `ANTHROPIC_API_KEY`
5. Deploy

Railway gives you $5/month free credit on the Trial plan. This bot uses ~$2–3/month worth, so you're well within the free tier.

### Persistence Note

By default, user prefs save to `./data/users.json`. Railway's default filesystem is **ephemeral** — on redeploy, prefs reset to defaults. For your sister's single-user case this is fine (she'll just `/start` again, defaults are sensible). If you want persistence, add a Railway Volume (paid add-on) and set `DB_PATH=/data/users.json`.

## Bot Commands

| Command | Description |
|---------|-------------|
| `/start` | Onboarding, sets defaults |
| `/today` | Send current affairs immediately |
| `/quiz` | Instant 5-question MCQ practice |
| `/a1`, `/a2`, ... | Reveal answer for quiz question N |
| `/settime HH:MM` | Set daily delivery time (24h IST) |
| `/setcount N` | News items per digest (5–15) |
| `/setmcqs N` | MCQs per digest (0–10) |
| `/pause` | Stop daily messages |
| `/resume` | Restart daily messages |
| `/status` | Show current settings |
| `/help` | Command list |

## Defaults (when sister runs `/start`)

- Time: **07:00 IST**
- News count: **10**
- MCQ count: **5**

She can change all via commands. Recommend starting with these and adjusting after a couple days based on her schedule.

## Telegram Tips for Sister

1. Open Telegram → search bot username (the one you created with @BotFather)
2. Tap **Start** → onboarding message will explain everything
3. Pin the chat to top so morning notifications are easy to find
4. After `/today` or `/quiz`, MCQs come as separate messages — just tap `/a1` etc. in chat to see answers
5. Bot is silent except at her chosen time + when she runs commands

## Troubleshooting

**Bot not responding** → Check Railway logs, ensure `TELEGRAM_BOT_TOKEN` is set
**No MCQs in messages** → `ANTHROPIC_API_KEY` missing or out of credits
**News not coming** → GKToday RSS occasionally rate-limits; bot has HTML fallback
**Time not matching IST** → Bot uses `Asia/Kolkata` regardless of server location, so this should always work
